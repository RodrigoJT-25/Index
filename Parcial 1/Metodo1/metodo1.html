Método : Bisección

Regresar Metodo Ejemplo

1. Definición

El método de bisección es un algoritmo numérico que permite encontrar una raíz real de una función continua f(x) en un intervalo cerrado [a, b], donde la función cambia de signo (es decir, f(a)·f(b) < 0). Se basa en dividir iterativamente el intervalo a la mitad, buscando el subintervalo donde ocurre el cambio de signo.

2. Antecedentes

Este método es uno de los más antiguos conocidos para resolver ecuaciones, usado desde la época de los griegos. Fue formalizado matemáticamente en el siglo XVII. Aunque tiene una convergencia lenta, es valorado por su simplicidad, robustez y certeza de convergencia.

3. Relación con Otros Métodos

Método de la falsa posición (regla falsa): Similar en estructura, pero ajusta el punto de evaluación usando una recta secante.

Método de Newton-Raphson: Utiliza derivadas para aproximar raíces con mayor rapidez, pero requiere condiciones específicas.

Método de la secante: Alternativa al método de Newton que no necesita derivadas.

Todos estos son métodos iterativos para encontrar raíces de funciones no lineales.

4. Fórmula Matemática

Para un intervalo [a, b] tal que f(a)·f(b) < 0, se define:

c = (a + b) / 2

Luego se evalúa f(c). Dependiendo del signo:

  • Si f(a)·f(c) < 0, la raíz está en [a, c]
  • Si f(c)·f(b) < 0, la raíz está en [c, b]
  • Si f(c) = 0, se ha encontrado la raíz exacta

Se repite hasta que |b - a| < ε (tolerancia deseada).

5. Algoritmo Paso a Paso

  1. Elegir un intervalo [a, b] donde f(a)·f(b) < 0
  2. Calcular c = (a + b) / 2
  3. Evaluar f(c)
  4. Si |f(c)| < ε, c es la raíz
  5. Si f(a)·f(c) < 0, b = c
  6. Si f(c)·f(b) < 0, a = c
  7. Repetir hasta converger

6. Aplicaciones

  • Resolución de ecuaciones no lineales
  • Ingeniería: encontrar puntos de equilibrio
  • Economía: intersecciones de oferta y demanda
  • Situaciones donde la derivada no está disponible
