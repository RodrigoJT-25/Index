Método de Euler

Definición

El Método de Euler es un procedimiento numérico para aproximar la solución de una ecuación diferencial ordinaria (EDO) de la forma:
dy/dx = f(x, y),   y(x₀) = y₀
Es un método explícito y de primer orden que utiliza la pendiente en el punto actual para estimar el valor de la función en el siguiente punto.

Antecedentes y métodos relacionados

  • Propuesto por Leonhard Euler en el siglo XVIII.
  • Derivado de la fórmula de aproximación lineal de una función.
  • Relacionado con: Método de Taylor, Método de Runge-Kutta, Método del Trapecio.

Fórmula matemática

yₙ₊₁ = yₙ + h · f(xₙ, yₙ)

Algoritmo

  1. Establecer las condiciones iniciales: x₀, y₀, paso h, número de pasos N.
  2. Para cada n = 0, 1, …, N-1:
    • Calcular yₙ₊₁ = yₙ + h · f(xₙ, yₙ)
    • Avanzar xₙ₊₁ = xₙ + h

Aplicaciones en la vida cotidiana

  • Simular el crecimiento de poblaciones.
  • Predecir temperaturas en un sistema físico.
  • Calcular trayectorias en física y astronomía.
  • Modelar procesos químicos simples.
  • Usado en computación y gráficos para integrar movimientos.

Método de Euler Modificado (o del Punto Medio)

Definición

El Método de Euler Modificado, también llamado método de punto medio o Heun, es una mejora del método de Euler. Usa una pendiente promedio entre el inicio y un punto estimado al final del intervalo.

Antecedentes y métodos relacionados

  • Desarrollado posteriormente a Euler para mayor precisión.
  • Relacionado con: Método del Trapecio, Método de Runge-Kutta.

Fórmula matemática

Primero se calcula un predictor:
y* = yₙ + h · f(xₙ, yₙ)
Luego se corrige tomando el promedio de las pendientes:
yₙ₊₁ = yₙ + (h/2) [f(xₙ, yₙ) + f(xₙ₊₁, y*)]

Algoritmo

  1. Establecer las condiciones iniciales: x₀, y₀, paso h, número de pasos N.
  2. Para cada n = 0, 1, …, N-1:
    • Calcular predictor: y* = yₙ + h · f(xₙ, yₙ)
    • Calcular corrector: yₙ₊₁ = yₙ + (h/2) [f(xₙ, yₙ) + f(xₙ₊₁, y*)]
    • Avanzar xₙ₊₁ = xₙ + h

Aplicaciones en la vida cotidiana

  • Simulación de circuitos eléctricos.
  • Modelos de propagación de enfermedades.
  • Problemas de dinámica en ingeniería.
  • Cálculos de trayectorias en mecánica y robótica.
  • Útil cuando la pendiente cambia rápidamente.
