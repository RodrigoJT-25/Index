Método de Simpson

Definición del método

El método de Simpson es una técnica de integración numérica que aproxima el valor de una integral definida ∫a^b f(x) dx usando polinomios interpolantes.

  • El método de Simpson 1/3 usa un polinomio de segundo grado (parábola) para aproximar la curva de f(x) en cada subintervalo.
  • El método de Simpson 3/8 usa un polinomio de tercer grado (cúbica) para mejorar la aproximación en ciertos casos.

Antecedentes del método y con qué otros métodos se relaciona

Estos métodos derivan del trabajo de Thomas Simpson (1710–1761), aunque ya habían sido descubiertos anteriormente por matemáticos como Newton y Cotes.

Se relacionan con:

  • Regla del trapecio (grado 1)
  • Cuadratura de Newton-Cotes
  • Métodos de integración adaptativa
  • Métodos de Romberg y cuadratura de Gauss

Fórmulas matemáticas

Método de Simpson 1/3:

Para una integral en [a,b] con un número par de subintervalos n: 

∫a^b f(x) dx ≈ (h/3) [ f(x₀) + 4 Σf(xᵢ) (i impares) + 2 Σf(xᵢ) (i pares) + f(xₙ) ]

donde h = (b-a)/n y xᵢ = a + i·h.

Método de Simpson 3/8:

Para una integral en [a,b] con un número de subintervalos n múltiplo de 3: 

∫a^b f(x) dx ≈ (3h/8) [ f(x₀) + 3 Σf(xᵢ) (i ≠ múltiplo de 3) + 2 Σf(xᵢ) (i múltiplo de 3) + f(xₙ) ]

donde h = (b-a)/n y xᵢ = a + i·h.

Algoritmo

Para Simpson 1/3:

  1. Dividir el intervalo [a,b] en un número par n de subintervalos.
  2. Calcular h = (b-a)/n.
  3. Evaluar f(x) en los puntos x₀, x₁, ..., xₙ.
  4. Aplicar la fórmula del método 1/3.
  5. Sumar los términos y multiplicar por h/3.

Para Simpson 3/8:

  1. Dividir el intervalo [a,b] en un número n de subintervalos múltiplo de 3.
  2. Calcular h = (b-a)/n.
  3. Evaluar f(x) en los puntos x₀, x₁, ..., xₙ.
  4. Aplicar la fórmula del método 3/8.
  5. Sumar los términos y multiplicar por 3h/8.

Aplicaciones en la vida cotidiana

  • Cálculo de áreas o volúmenes cuando la función no tiene una primitiva sencilla (por ejemplo, mediciones experimentales).
  • Ingeniería civil: para calcular áreas bajo curvas de esfuerzo-deformación o de caudales en ríos.
  • Física: para estimar trabajo realizado cuando la fuerza varía no linealmente.
  • Economía: para calcular áreas bajo curvas de costos o ingresos.
  • Ciencias ambientales: para estimar el área cubierta por una especie en un ecosistema.
