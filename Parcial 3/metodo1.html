Método de Diferencias Divididas

Regresar Ejemplo Método

Definición del método

El método de diferencias divididas de Newton es un procedimiento para encontrar un polinomio que interpola un conjunto de puntos dados (x₀, y₀), (x₁, y₁), …, (xₙ, yₙ). Este método utiliza el concepto de diferencias divididas, que son coeficientes calculados de forma recursiva a partir de los valores de las funciones y de las abscisas. Es especialmente útil cuando los puntos no están igualmente espaciados.

Antecedentes del método y con qué otros métodos se relaciona

El método fue desarrollado por Isaac Newton en el siglo XVII como parte de su trabajo en análisis numérico. Está estrechamente relacionado con otros métodos de interpolación polinómica, como:

  • Interpolación de Lagrange
  • Método de diferencias finitas
  • Interpolación spline

Fórmula matemática que lo define

El polinomio interpolante de Newton con diferencias divididas se expresa como:

P(x) = f[x₀] + f[x₀,x₁](x-x₀) + f[x₀,x₁,x₂](x-x₀)(x-x₁) + … + f[x₀,…,xₙ](x-x₀)(x-x₁)…(x-xₙ₋₁)

Donde las diferencias divididas se calculan recursivamente:

f[x₀] = y₀
f[x₀,x₁] = (f[x₁] - f[x₀]) / (x₁ - x₀)
f[x₀,x₁,x₂] = (f[x₁,x₂] - f[x₀,x₁]) / (x₂ - x₀) y así sucesivamente.

Algoritmo

  1. Ingresar los valores xᵢ y yᵢ.
  2. Inicializar la primera columna con yᵢ.
  3. Para cada orden k, calcular:
    f[xᵢ, ..., xᵢ₊ₖ] = (f[xᵢ₊₁, ..., xᵢ₊ₖ] - f[xᵢ, ..., xᵢ₊ₖ₋₁]) / (xᵢ₊ₖ - xᵢ)
  4. Repetir hasta completar todas las diferencias.
  5. Formar el polinomio con los coeficientes obtenidos.

Aplicación en la vida cotidiana

El método de diferencias divididas tiene aplicaciones en:

  • Ingeniería: para aproximar funciones complejas cuando solo se conocen valores discretos.
  • Física: para interpolar mediciones experimentales y estimar valores intermedios.
  • Economía: para construir modelos predictivos a partir de datos históricos.
  • Computación gráfica: para calcular curvas y superficies suaves a partir de puntos de control.
  • Astronomía: para calcular órbitas o trayectorias de cuerpos celestes a partir de observaciones.
