MÉTODO DE JACOBI

1. Definición

El método de Jacobi es un método iterativo usado para resolver sistemas de ecuaciones lineales del tipo Ax = b. Cada componente del nuevo vector de soluciones se calcula usando solo los valores de la iteración anterior.

2. Antecedentes y Métodos Relacionados

  • Desarrollado por Carl Gustav Jacobi en el siglo XIX.
  • Relacionado con:
    • Método de Gauss-Seidel
    • Métodos directos como Eliminación de Gauss
    • Métodos iterativos modernos

3. Fórmula Matemática

x(k+1) = D−1 ( b − (L+U) x(k) )

4. Algoritmo Paso a Paso

  1. Reescribir el sistema aislando cada variable.
  2. Elegir una aproximación inicial x^(0).
  3. Calcular todas las variables usando la fórmula de Jacobi.
  4. Actualizar todas al final de cada iteración.
  5. Repetir hasta alcanzar el error deseado.

5. Aplicaciones en la Vida Cotidiana

  • Modelado computacional de flujos de calor.
  • Problemas estructurales en ingeniería.

MÉTODO DE GAUSS-SEIDEL

1. Definición

El método de Gauss-Seidel también resuelve Ax = b, pero mejora la eficiencia usando los valores más recientes conforme se calculan.

2. Antecedentes y Métodos Relacionados

  • Nombrado por Carl Friedrich Gauss y Philipp Seidel.
  • Relacionado con:
    • Método de Jacobi
    • Método SOR (Successive Over Relaxation)
    • Métodos modernos iterativos

3. Fórmula Matemática

x(k+1) = (D+L)−1 ( b − U x(k) )

4. Algoritmo Paso a Paso

  1. Reescribir el sistema aislando cada variable.
  2. Establecer una estimación inicial.
  3. Para cada variable, usar la fórmula con valores actualizados.
  4. Repetir hasta converger.

5. Aplicaciones en la Vida Cotidiana

  • Simulación de estructuras físicas.
  • Análisis de circuitos eléctricos.
  • Simulación de fluidos y calor.
